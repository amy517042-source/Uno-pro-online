# UNO Pro Online

A professional multiplayer UNO game built with:

- ⚛️ React
- ⚡ Vite
- 🎨 Tailwind CSS
- 🔥 Firebase
- ☁️ Vercel

## Features

- Online Multiplayer
- Firebase Authentication
- Firestore Realtime Database
- AI Strategy Assistant
- Responsive Design
- Professional UI

## Tech Stack

- React
- Vite
- Tailwind CSS
- Firebase
- Vercel

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
